# Age-related-disease-classification-model
The goal of this analysis is to predict using randomforest classifier and xgboost algorithms if a person has any of three medical conditions related to aging. 
Data was downloaded from the competition's official page at: https://www.kaggle.com/competitions/icr-identify-age-related-conditions/data.
This notebook keeps being updated as I analyse and discover new ways to optimise the ML models.
